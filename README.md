# NetSlicer — Subnet Calculator

A modern, cyberpunk-themed subnet calculator for network engineers and IT professionals.

## Features

- **Subnet Calculator** — Calculate subnet details from IP/CIDR input
- **VLSM Calculator** — Variable Length Subnet Masking for efficient address allocation
- **IPv6 Calculator** — IPv6 subnet calculations
- **Wildcard Mask Calculator** — Calculate wildcard masks from subnet masks
- **Binary Converter** — Convert between decimal and binary IP representations
- **Subnet List** — View all subnets in a given supernet

## Usage

Simply open `subnet-calculator.html` in any modern web browser. No server or installation required.

### Input Formats

- **CIDR notation**: `192.168.1.0/24`
- **Subnet mask**: `192.168.1.0 255.255.255.0`

## Tech

- Single HTML file with embedded CSS/JS
- Tailwind CSS (via CDN)
- Custom cyberpunk/neon UI theme
